### 입력 

 <p>첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.</p>

### 출력 

 <p>첫째 줄부터 2×N-1번째 줄까지 차례대로 별을 출력한다.</p>

### 배운점
- [*range(1, n+1), *range(n-1, 0, -1)] 를 통해서 range를 분해하고 다시 리스트로 합쳐서 범위를 합칠 수 있다.
- {변수명:<{n}} 이런식으로 n 자릿수에서 왼쪽 정렬을 구현할 수 있고 {변수명:>{n}} 로 n 자릿수에서 오른쪽 정렬, {변수명:^{n}} 로 n 자릿수에서 가운데 정렬을 구현할 수 있다.
