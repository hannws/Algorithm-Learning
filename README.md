# Algorithm Study (Baekjoon)

백준 알고리즘 문제 풀이 및 사고 과정을 기록하는 저장소입니다.  
**백준허브(BaekjunHub)** 를 사용하여 자동으로 소스코드를 업로드하고, 상세한 풀이 과정을 정리합니다.

---

## 환경 및 도구
- **언어**: Python 3 / C++
- **자동 업로드**: [BaekjunHub](https://github.com/BaekjunHub/BaekjunHub)
- **문제 링크**: [Baekjoon Online Judge](https://www.acmicpc.net/)

---

## 저장소 구조
- 문제 풀이 코드는 백준허브를 통해 자동으로 폴더링됩니다.
- 각 문제 폴더 안의 `README.md`에 해당 문제에 대한 **사고 방식**과 **풀이 전략**을 기록합니다.

---

## 나만의 풀이 정리 규칙
문제를 풀고 난 후, 아래 항목들을 중점적으로 정리합니다.

1. **아이디어**: 문제를 처음 보고 생각한 접근법
2. **핵심 로직**: 문제 해결을 위한 알고리즘 및 시간 복잡도 분석
3. **트러블 슈팅**: 발생했던 에러나 시간 초과 해결 과정
4. **배운 점**: 새롭게 알게 된 문법이나 테크닉
